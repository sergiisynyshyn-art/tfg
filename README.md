# 🌿 Serenum — Aplicación de Meditación y Bienestar

Serenum es una aplicación móvil desarrollada como proyecto final del ciclo de **Desarrollo de Aplicaciones Multiplataforma (DAM)**.  
Su objetivo es ofrecer una herramienta sencilla, accesible y moderna para ayudar a los usuarios a reducir el estrés, mejorar la concentración y crear hábitos saludables mediante prácticas de meditación, sonidos relajantes y rutinas guiadas.

Este proyecto combina diseño, experiencia de usuario, autenticación híbrida, base de datos local y reproducción multimedia, integrando conocimientos adquiridos durante el curso junto con tecnologías nuevas aprendidas durante el desarrollo.

---

## ✨ Características principales

- 🔐 **Autenticación híbrida**
  - Registro y login mediante email y contraseña
  - Inicio de sesión con **Google Sign‑In**
  - Contraseñas cifradas con SHA‑256

- 🗄️ **Base de datos local SQLite**
  - Gestión de usuarios
  - Favoritos
  - Progreso y rutinas

- 🎧 **Contenido multimedia**
  - Meditaciones guiadas
  - Sonidos relajantes
  - Reproductor integrado con control de audio

- 🎨 **Diseño intuitivo y minimalista**
  - Pantallas limpias y orientadas al bienestar
  - Onboarding inicial con objetivos del usuario
  - Menú inferior con navegación clara

- ⭐ **Funciones adicionales**
  - Recomendación del día
  - Sistema de favoritos
  - Encuesta inicial de intención (Initial User Intent Survey)
  - Envío automático de correos de bienvenida (JavaMail)
